The return of the portfolio (\(R_p\)) is given by the weighted sum of the returns of individual assets:

\[
R_p = \sum_{i=1}^N W_i R_{A_i}
\]

where:
\begin{itemize}
    \item \(R_p\): Retorno del Portafolio
    \item \(N\): Numero total de activos en el portafolio
    \item \(A_i\): El Activo \(i\) 
    \item \(R_{A_i}\): Retorno del Activo i \(A_i\).
    \item \(W_i\): El peso del presupuesto asignado al Activo \(A_i\), con \(\sum_{i=1}^N W_i = 1\).
\end{itemize}

\[
\rho = \left( \rho_{i,j} \right)_{i,j=1,...,N}
\]

\[
\rho_{i,j} = \rho[X_i, X_j]
\]

\[
d : (X_i, X_j) \subseteq \mathcal{B} \to \mathbb{R} \in [0, 1], \quad d_{i,j} = d[X_i, X_j] = \sqrt{\frac{1}{2}(1 - \rho_{i,j})}.
\]


\[
\tilde{d} : (D_i, D_j) \subseteq \mathcal{B} \to \mathbb{R} \in [0, \sqrt{N}], \quad \tilde{d}_{i,j} = \tilde{d}[D_i, D_j] = \sqrt{\sum_{n=1}^N (d_{n,i} - d_{n,j})^2}.
\]
